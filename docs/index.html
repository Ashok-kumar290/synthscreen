Track 1

SynthGuard — Screening Engine

Lightweight ML triage combining ESM-2 protein embeddings (650M, LoRA fine-tuned) and a K-mer Random Forest for DNA. Targets the short-sequence false-positive problem and AI-designed variant blindspot in existing tools.

Quick start

 terminal 
# Clone & run with Docker (recommended)
git clone https://github.com/Ashok-kumar290/funcscreen
cd funcscreen
docker compose up --build

# Access the dashboard
open http://localhost:8501

# Or run directly (Python 3.11+)
pip install -r requirements.txt
streamlit run app.py -- --mode demo
